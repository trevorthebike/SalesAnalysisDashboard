Sales Analysis Dashboard

An interactive dashboard designed to visualize revenue, profit, and sales trends. This project transforms raw data into actionable business insights through structured analysis and interactive reporting.
Data Source

This dashboard uses the Rossmann Store Sales dataset, leveraging historical data across multiple stores, product categories, and timeframes to model revenue and performance.
How to Run

    Clone the repository: git clone https://github.com/trevorthebike/SalesAnalysisDashboard.git

    Install dependencies: pip install -r requirements.txt

    Launch the dashboard: streamlit run app.py
